About This Project

The Blinkit Seller Center is a comprehensive Django 6.0 application that provides both REST and SOAP APIs for managing various aspects of an e-commerce seller platform. The project is organized into domain-specific apps, each handling a specific business function.

Key Features

  • Complete REST API implementation using Django REST Framework
  • SOAP API endpoints with WSDL generation
  • Advanced filtering, searching, and pagination
  • Comprehensive admin interface
  • Well-structured and modular codebase
  • Detailed API documentation
