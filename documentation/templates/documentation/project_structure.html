Database Models

The project includes 12 database models across 7 apps:

Products App

  • Product: Product catalog with SKU, pricing, and inventory status

Orders App

  • Order: Order information and customer details
  • OrderItem: Individual items in an order

Inventory App

  • Inventory: Stock levels and warehouse locations

Catalog App

  • Category: Product categories with hierarchy
  • Brand: Product brands

Seller App

  • Seller: Seller profiles with GSTIN/PAN validation

Analytics App

  • SalesReport: Daily sales metrics
  • ProductPerformance: Product-level analytics

Documentation App

  • APIEndpoint: API endpoint documentation
  • Changelog: Version history and changes
  • SetupStep: Setup instructions

API Architecture

REST API

Built using Django REST Framework with the following features:

  • ViewSets for CRUD operations
  • Model serializers for data validation
  • Advanced filtering via django-filter
  • Search across multiple fields
  • Custom actions for business logic
  • Pagination support

SOAP API

Custom implementation using lxml with:

  • SOAP 1.1 protocol
  • WSDL generation
  • XML request/response handling
  • Fault handling
